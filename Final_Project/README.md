
## IT 3203 Cybersecurity Website Project

This project is a responsive multi-page website focused on cyber threats and cybersecurity concepts.

### Topics Covered:
- Malware
- Phishing
- DDoS Attacks
- Security principles
- Protection methods

### Features:
- Multi-page navigation
- Responsive design using CSS media queries
- Flexbox layout
- Cybersecurity educational content

### Testing:
- Desktop: Full layout with horizontal navigation
- Tablet: Adjusted layout
- Mobile: Stacked navigation for usability
