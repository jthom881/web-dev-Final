# IT 3203 Cybersecurity Website Project

## Project Overview

This project is a responsive multi-page website focused on cybersecurity concepts and common cyber threats. The purpose of this website is to educate users about major security risks, protection methods, and foundational cybersecurity principles using a structured and mobile-friendly web design.

The website was developed using HTML and CSS and deployed using GitHub Pages.

---

## Cybersecurity Topics Covered

This website includes educational content on the following cybersecurity topics:

- Malware
- Phishing
- DDoS Attacks
- Cybersecurity protection methods
- Core security principles (Confidentiality, Integrity, Availability)

Each topic is explained using structured content and organized into separate web pages for clarity and usability.

---

## Website Features and Design

This project follows modern responsive web design principles:

### Responsive Design
The website is fully responsive and adapts to different screen sizes (desktop, tablet, and mobile) using CSS media queries.

### Layout System
Flexbox is used to structure page layouts and ensure consistent alignment across all devices.

### Consistent Styling
A single external CSS file is used across all pages to maintain consistent:
- Colors
- Typography
- Layout structure
- Navigation design

### User Experience
The navigation menu is consistent across all pages, allowing easy movement between sections.

## Testing Report

The website was tested under multiple screen sizes:

### Desktop View
